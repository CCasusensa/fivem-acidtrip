# fivem-acidtrip
Trip balls in FiveM.

# Requirements
- FiveM/CFX
- No framework required.

# Installation

- Extract the `acidtrip` folder into your `resources` directory.
- Add `start acidtrip` in your server-cfg.

# Usage
`exports["acidtrip"]:DoAcid(120000)` for a two minute trip.
